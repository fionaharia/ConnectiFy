## **ConnectiFy - where every click connects you!**

_A social media fullstack application made using MERN stack_

### **Features**
1. Authorization
2. API Handling
3. Dark and Light Mode
4. Add a Friend List
5. Like Post
6. Image Upload

### **Tools Used**
1. MongoDB
2. React
3. Express
4. Node
5. Redux Toolkit
6. Material UI Library
7. Yup
8. Formik
